Current Data Collection Answer

For the current app code, the recommended App Store Connect answer is: No, we do not collect data from this app.

Reason: the current app stores library data locally, stores provider passwords and parental PINs in Keychain, has no app account, has no developer-operated backend, has no advertising SDK, has no analytics SDK, and does not transmit user library data to Gloud Tech Solutions.

Important Assumptions

  • No analytics, advertising, attribution, crash reporting, remote logging, or tracking SDK is added.
  • No cloud sync, account login, backend API, or support form is added inside the app.
  • The app continues to contact only user-entered playlist, guide, and provider URLs.
  • App Store screenshots and metadata do not claim that the app provides free TV, premium channels, sports, movies, providers, or playlists.

If The App Changes Later

Update App Store Connect before release if the app adds cloud backup, iCloud sync beyond private user storage, app accounts, analytics, crash reporting, subscriptions with server receipts, customer support forms, push notifications, advertising, or any third-party SDK that collects data.

These answers are product-preparation notes, not legal advice.
